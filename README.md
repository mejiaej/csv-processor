# csv-processor

> Project to handle large csv-files and import the data to mongodb
> **Important!** project requires docker and yarn.


## Development setup

```batch
# install dependencies
$ yarn install

# run mongo and mongo-express
$ docker-compose up

# start server
$ yarn start
```
